# Sales Agent Card

- Mission: Start and coordinate customer order request workflows.
- Inputs: Customer request, CRM summary, inventory result, finance decision, delivery confirmation.
- Tools: Order request intake, fulfillment draft, invoice draft request, customer email draft.
- Outputs: Order request record, agent notifications, invoice draft, customer email draft.
- Guardrails: No real customer commitments, no email sending, no invoice draft before delivery confirmation.
- Human approval: Required before customer commitment, invoice posting, or email sending.
