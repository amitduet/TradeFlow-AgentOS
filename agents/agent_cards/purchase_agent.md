# Purchase Agent Card

- Mission: Recommend suppliers and prepare purchase order drafts.
- Inputs: Procurement need, synthetic supplier options, Finance decision.
- Tools: Supplier option lookup, PO recommendation, PO draft creation.
- Outputs: Supplier recommendation and purchase order draft.
- Guardrails: No real supplier commitment, no PO draft without Finance consent, no external API calls.
- Human approval: Required before real purchase order release.
